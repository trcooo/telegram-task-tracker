Память задач (почему они пропали)
1) На Railway контейнер может пересоздаваться. Если использовалась SQLite без внешнего Postgres — задачи "сбрасываются".
   Решение: подключить Railway Postgres и выставить DATABASE_URL в переменные окружения.
2) Раньше в браузере задачи создавались под user_id=1, а в Telegram user_id другой — поэтому "пусто".
   В этой версии добавлен перенос: при первом входе из Telegram спросит про перенос задач.
